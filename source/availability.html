Tillgänglighet

Vi på Färdtjänsthjälpen står bakom denna webbsida. Vi vill att så många som möjligt ska kunna använda webbplatsen och därför försöker vi tillgänglighetsgöra sidan. Här kan du läsa om våran redogörelse för hur vi arbetar med tillgänglighet på webbsidan.

Hur arbetar vi med tillgänglighet?

Någonting om arbetet

Med färdtjänst kan du resa inom hela Örebro Län. Du kan dessutom resa utanför länet men då är begränsningen 30 km från länsgränsen och resan ska börja eller sluta inom Örebro län. Om resan är längre än så kan du ansöka om riksfärdtjänst via Örebro Kommuns blanketter alternativt via dina vårdtjänster på 1177.
